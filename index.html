Introdução

Bem-vindo à documentação oficial do Fluent v2. Esta biblioteca oferece uma suíte de interface moderna para Roblox, com suporte a temas, animações e componentes profissionais. Aprenda passo a passo como usar cada parte da lib, como em um tutorial guiado.

Versão: 2.0.0. Modificado para tema AMOLED vermelho, scrollbars vermelhas e mais.

Próximo: Instalação >

Instalação

Comece instalando o Fluent v2 em seu executor Roblox. Use o loadstring para carregar o script remotamente de forma segura.

loadstring(game:HttpGet("https://sua-url-do-script-aqui.com/fluentv2.lua"))()

Certifique-se de usar um executor confiável como Fluxus. Após carregar, a lib estará disponível como 'Fluent'.

Criando uma Janela

Crie uma janela principal com opções personalizadas. Exemplo básico:

local window = Fluent.CreateWindow({
    Title = "Minha UI",
    Size = UDim2.fromOffset(580, 460),
    Acrylic = true,
    Theme = "Dark"
})

Isso inicia uma janela com tema Dark e efeitos acrylic. Use window para adicionar tabs e elementos.

Adicionando Tabs

Adicione tabs à janela para organizar conteúdo. Exemplo:

local tab = window:AddTab({
    Title = "Configurações",
    Icon = "lucide-settings"
})

Selecione tabs com window:SelectTab(index). Tabs suportam ícones da lib integrada.

Adicionando Seções e Elementos

Crie seções em tabs e adicione elementos como buttons, toggles, etc. Exemplo:

local section = tab:AddSection("Opções Gerais")
section.Container:Button({
    Title = "Ativar",
    Callback = function() print("Botão clicado") end
})
section.Container:Toggle("toggle1", {
    Title = "Ativar Cheat",
    Default = false,
    Callback = function(value) print("Toggle:", value) end
})

Outros elementos: Slider, Dropdown, Colorpicker, Keybind, Input. Cada um retorna objetos para controle de valor.

Temas

Altere o tema da UI dinamicamente. Temas disponíveis: Dark, Darker, Light, Aqua, Amethyst, Rose.

Fluent:SetTheme("Dark")

O tema Dark usa AMOLED black com acentos vermelhos. Personalize cores no objeto Fluent.Themes.

Notificações e Diálogos

Mostre notificações e diálogos interativos. Exemplo de notificação:

Fluent:Notify({
    Title = "Sucesso",
    Content = "UI Carregada",
    Duration = 5
})

Para diálogos: window:Dialog({Title = "Confirmação", Content = "Continuar?", Buttons = {{Title = "Sim", Callback = function() end}}})

Features Avançadas

Explore features como minimizar (window:ToggleMinimize()), maximizar (window:Maximize(true)), e suporte a keybinds (Fluent.MinimizeKey = Enum.KeyCode.LeftControl).

Integre com SaveManager e InterfaceManager para salvar configs e personalizações.
