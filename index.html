Introdução

Bem-vindo à documentação oficial do Fluent v2. Esta biblioteca oferece uma suíte de interface moderna para Roblox, com suporte a temas, animações e componentes profissionais. Este tutorial guiará você passo a passo, como em um mangá, para dominar cada aspecto da biblioteca.

Versão: 2.0.0. Tema AMOLED vermelho com scrollbars neon e cards glassmorphic.

Instalação

Instale o Fluent v2 em seu executor Roblox usando loadstring para carregar o script remotamente:

loadstring(game:HttpGet("https://sua-url-do-script-aqui.com/fluentv2.lua"))()

Use um executor confiável como Fluxus. Após carregar, a biblioteca estará disponível como 'Fluent'.

Criando uma Janela

Crie uma janela principal com opções personalizadas:

local window = Fluent.CreateWindow({
    Title = "Minha UI",
    Size = UDim2.fromOffset(580, 460),
    Acrylic = true,
    Theme = "Dark"
})

Isso cria uma janela com tema Dark e efeitos acrylic. Use 'window' para adicionar tabs e elementos.

Adicionando Tabs

Adicione tabs à janela para organizar o conteúdo:

local tab = window:AddTab({
    Title = "Configurações",
    Icon = "lucide-settings"
})

Selecione tabs com window:SelectTab(index). Ícones são suportados pela biblioteca integrada.

Adicionando Seções e Elementos

Crie seções em tabs e adicione elementos como botões e toggles:

local section = tab:AddSection("Opções Gerais")
section.Container:Button({
    Title = "Ativar",
    Callback = function() print("Botão clicado") end
})
section.Container:Toggle("toggle1", {
    Title = "Ativar Cheat",
    Default = false,
    Callback = function(value) print("Toggle:", value) end
})

Outros elementos disponíveis: Slider, Dropdown, Colorpicker, Keybind, Input.

Temas

Altere o tema da UI dinamicamente. Temas disponíveis: Dark, Darker, Light, Aqua, Amethyst, Rose.

Fluent:SetTheme("Dark")

O tema Dark usa AMOLED black com acentos vermelhos. Personalize cores em Fluent.Themes.

Notificações e Diálogos

Mostre notificações e diálogos interativos:

Fluent:Notify({
    Title = "Sucesso",
    Content = "UI Carregada",
    Duration = 5
})

Para diálogos: window:Dialog({Title = "Confirmação", Content = "Continuar?", Buttons = {{Title = "Sim", Callback = function() end}}})

Features Avançadas

Explore recursos como minimizar (window:ToggleMinimize()), maximizar (window:Maximize(true)), e keybinds (Fluent.MinimizeKey = Enum.KeyCode.LeftControl).

Integre com SaveManager e InterfaceManager para salvar configurações.
