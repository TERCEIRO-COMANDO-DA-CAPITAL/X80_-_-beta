Introdução

Bem-vindo à documentação oficial do Fluent v2, uma suíte de interface avançada para Roblox projetada para desenvolvedores profissionais. O Fluent v2 oferece uma biblioteca de UI moderna com temas personalizáveis, suporte a acrylic e glassmorphism, animações suaves e componentes intuitivos.

Autor: dawid (modificado por Grok para tema AMOLED, acentos vermelhos, scrollbars vermelhas e botão de minimizar). Licença: MIT. GitHub: https://github.com/dawid-scripts/Fluent.

Versão: 2.0.0. Esta biblioteca é otimizada para performance e compatibilidade com as versões mais recentes do Roblox.

Instalação

Para instalar o Fluent v2, você precisa de um executor de scripts compatível com Roblox, como Fluxus, Synapse ou similar. Siga os passos abaixo:

  1. Instale um executor de confiança no seu dispositivo.
  2. Abra o Roblox e injete o executor.
  3. Execute o loadstring fornecido abaixo, substituindo a URL pelo link do seu script hospedado.
loadstring(game:HttpGet("https://sua-url-do-script-aqui.com/fluentv2.lua"))()

Após a execução, a biblioteca será carregada automaticamente. Certifique-se de que a URL aponta para o arquivo .lua hospedado de forma segura (ex: GitHub Raw ou Pastebin).

Uso

O Fluent v2 é fácil de usar. Aqui vai um guia básico para criar uma janela e adicionar elementos:

  • Criar Janela: Use Fluent.CreateWindow({Title = "Minha UI", Size = UDim2.fromOffset(580, 460)})
  • Adicionar Tab: window:AddTab({Title = "Configurações", Icon = "lucide-settings"})
  • Adicionar Seção: tab:AddSection("Opções Gerais")
  • Adicionar Elementos: section.Container:Button({Title = "Ativar", Callback = function() print("Ativado") end})

Exemplo de loadstring para versão personalizada (substitua a URL):

loadstring(game:HttpGet("https://sua-url-personalizada-aqui.com/fluentv2-custom.lua"))()

Para notificações: Fluent:Notify({Title = "Sucesso", Content = "UI Carregada", Duration = 5})

Para diálogos: window:Dialog({Title = "Confirmação", Content = "Continuar?", Buttons = {{Title = "Sim", Callback = function() end}}})

Features

O Fluent v2 inclui uma ampla gama de features avançadas:

  • Temas Personalizáveis: Dark, Darker, Light, Aqua, Amethyst, Rose - com acentos vermelhos no Dark.
  • Acrylic e Glassmorphism: Suporte a efeitos de blur e transparência para um visual moderno.
  • Animações Suaves: Usando Flipper para springs, lineares e instantâneas.
  • Notificações: Sistema de notificações com duração e animações.
  • Botão de Minimizar: Adicionado para alternar visibilidade da UI.
  • Scrollbars Vermelhas: Customizadas para o tema AMOLED.
  • Ícones Lucide: Biblioteca de ícones integrada.
  • Componentes: Botões, Toggles, Sliders, Dropdowns, Colorpickers, Keybinds, Inputs, etc.
  • Suporte a Keybinds: Para minimizar a UI (padrão: LeftControl).

Para mais detalhes, consulte as seções de Temas e Elementos abaixo.

Temas

O Fluent v2 suporta múltiplos temas. Use Fluent:SetTheme("NomeDoTema") para alternar.

  • Dark (Padrão): AMOLED black com acentos vermelhos claros.
  • Darker: Tons mais escuros para ambientes low-light.
  • Light: Tema claro para visibilidade diurna.
  • Aqua: Tons de azul para um visual fresco.
  • Amethyst: Tons roxos elegantes.
  • Rose: Tons rosados vibrantes.

Exemplo: Fluent:SetTheme("Dark")

Elementos

Adicione elementos às seções para criar interfaces ricas:

  • Button: Adiciona um botão clicável. Ex: section.Container:Button({Title = "Clique", Callback = function() end})
  • Paragraph: Texto descritivo. Ex: section.Container:Paragraph({Content = "Descrição aqui"})
  • Toggle: Alternador on/off. Ex: section.Container:Toggle("ID", {Title = "Ativar Cheat", Default = false, Callback = function(value) end})
  • Slider: Controle deslizante. Ex: section.Container:Slider("ID", {Title = "Velocidade", Default = 50, Min = 0, Max = 100, Callback = function(value) end})
  • Dropdown: Seleção de opções. Ex: section.Container:Dropdown("ID", {Title = "Escolha", Default = "Opção 1", Options = {"Opção 1", "Opção 2"}, Callback = function(value) end})
  • Colorpicker: Seletor de cor. Ex: section.Container:Colorpicker("ID", {Title = "Cor", Default = Color3.fromRGB(255, 100, 100), Callback = function(color) end})
  • Keybind: Atribuição de tecla. Ex: section.Container:Keybind("ID", {Title = "Bind", Default = Enum.KeyCode.E, Callback = function(key) end})
  • Input: Campo de texto. Ex: section.Container:Input("ID", {Title = "Nome", Default = "", Callback = function(text) end})

Todos os elementos suportam temas e transparências para um visual coeso.
